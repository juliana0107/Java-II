# Proyecto Final java avanzado

### Iniciar el proyecto
Si usted desea ejecutar este proyecto en un servidor a travez de un docker-compose

Puede realizarlo con los siguientes comandos
* mvn clean install
* docker-compose build          
* docker-compose up
